Description

The server-side component of a “movies” web application.

Scope

The web application will provide users with access to information about different movies, directors, and genres.

Usage

Users will be able to:

  • sign up
  • update their personal information
  • create a list of their favorite movies

API Documentation

Description URL HTTP Method Query Parameters Request Response
Retrieve all movies /movies GET
Retrieve data about a movie by title /movies/:title GET
Retrieve data about a genre by name /movies/:genres/:genreName GET
Retrieve data about a director by name /movies/:directors/:directorName GET
Add user /users POST
Update user data /users/:id PUT
Add movie to favorites /movies/:id/:movieTitle POST
Remove movie from favorites /movies/:favorites DELETE
Delete user /users DELETE
